# AWS-IAM-Lab
AWS IAM lab in a live environment. Built users, groups, and a custom JSON least-privilege policy. Attached an IAM role to EC2 — verified S3 access with zero stored credentials. Enabled MFA via authenticator app. Covers all 5 core IAM components.
# AWS IAM Lab — Users, Groups, Roles, Policies & MFA

**Completed:** May 5, 2026  
**Platform:** AWS IAM (Live Environment)

## What I built
- IAM user (chase-dev-user) with console access
- IAM group (Developers) with PowerUserAccess
- IAM role (chase-s3-ec2-role) attached to EC2 — verified S3 access with no stored credentials
- Custom least-privilege JSON policy scoped to a single S3 bucket
- MFA enabled via authenticator app

## Skills demonstrated
AWS IAM · JSON Policy Authoring · Least-Privilege Access · EC2 Role Attachment · MFA · Cloud Security
